Ubuntu/Kali 安装谷歌拼音输入法

白帽子续命指南
没有肌肉的白帽子，进去了也是受欺负！

在这一切操作之前，请确保换成国内的软件源

1、更新一下

sudo apt update

2、安装fcitx

sudo apt install fcitx

3、安装fcitx-google

sudo apt install fcitx-googlepinyin

4、按win键，输入input method，回车

5、选择fcitx

6、重启

reboot

7、按win键，输入fcitx config，回车

8、左下角加号

9、左下角“对勾”取消，输入google，选择Google Pinyin，OK

10、此时就可以使用了，按Ctrl + Space切换输入法

11、也可以配置一下，因为我比较喜欢“，”“。”切换候选次。就和以前一直用的搜狗输入法一样

按win，输入fcitx config，回车

这个Gobal Config界面就可以设置了sudo apt install fcitx-googlepinyin
