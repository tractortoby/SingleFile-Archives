What is the difference between onchange and onclick in JavaScript?
6 Answers
Christian Schüler
, Software Developer (1999-present)
Answered Aug 29, 2018

The main difference between these two events is the trigger (when the event is fired). As all answers before mentioned, the trigger for onclick is the click on the element and the trigger for onchange is the changing of the value of the element.

As you can see, the onchange event can only be fired by every element that has a value property that can be changed by the user or another function. The onclick event can be fired by every element that is visibly rendered by the browser and the user can click on it.

Don’t mix it with the element that reacts on the event. Events in JavaScript are climbing up (bubbling up) the DOM tree and every node they pass can react on the event. So the event listener for the onchange event can be attached to a node without a value property. This node reacts only to the event and within the event arguments, you can determine which element was the target which fired the event. The onclick event listener can be attached to an element/node that is not visibly rendered and the user can’t click on it. It only reacts to the event and, as mentioned in the onchange eventlistener, you can determine which node fired the onclick event.

A short example for onchange (pseudo code):

  1. FORM-Element.addEventListener(‘click’, function(evt) { 
  2.     console.log(evt.target); 
  3. }) 

The form element itself doesn’t have a value property, but it can react to the event(s) fired by form input elements, like input, select, textbox, etc.

With the knowledge that every event is climbing up the DOM tree, you can define centralized event listener and don’t need to attach a listener to every element which fires the event. You only have to control that the event will pass the node you will attach the listener to.

onclick is an event that declares when something, anything, is clicked.

onchange is an event that declares when an input is changed.

If the event is a click that changes the input, then there is no difference. Both will be declared.
