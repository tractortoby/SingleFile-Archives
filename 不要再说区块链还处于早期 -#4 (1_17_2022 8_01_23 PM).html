区块链已不是早期#

其实最近讲 Web3 什么的有点说烦了，但实在是我的 timeline 里全是这些关键词，躲不过，就顺手又读了几篇，读完觉得还是有必要把想法写出来。

在区块链世界里，现在有很多门派，NFT 派，DeFi 派，Web3 派，Metaverse 派，我现在属于无门无派，可能倾向于原教旨主义的 Bitcoin 派，但就像我上一期说的，我对于新事物完全不排斥，只是仍在以审视的眼光看待它们。

其实质疑一个事物，不是表示不支持，反而是给它充分的改进机会，假如一个事物经不起推敲，那还有什么价值；又假如一个事物能经受住千锤百炼，那它必然就是真理。但不管怎样质疑币圈的朋友们，你都会听到一句话：“现在还是区块链的早期”，然后劝你跑步入场。听到早期我不会有太大反应，但拉我入场，我会下意识反射这是不是骗子（可能被骗多了，苦笑）。

近期读到 It's not still the early days，作者疾呼“现在已经不能再称为早期了”，我读罢深以为然。众所周知，科技发展遵循摩尔定律 —— 即处理器性能每隔 18 个月翻一倍。我们经常惊呼科技的发展速度，但置身于时间洪流时却忘记了它到底多快。Bitcoin 发迹于 2009 年，Ethereum 诞生于 2015 年，让我们看看这两年科技圈分别发生了什么：

2009#
  • iPhone 3GS 发布，没有物理按键的手机刚刚开始流行，移动互联网还没有大行其道
  • 人们刚开始用 Twitter，新浪微博 2009 年 8 月才推出
  • Elon Musk 刚刚接任 Tesla CEO 之后，第一次向世界展示了 Model S（三年后才产出上市）
  • Uber 刚刚成立，滴滴打车在三年后才出现
  • Windows 7 发布
  • Intel 处理器发布了酷睿 i5
  • 你喜欢的小破站 Bilibili 刚刚破壳
  • “双十一” 光棍节第一次变成购物节
  • 技术方面，JQuery 刚刚起飞，Node.js 首次发布，MongoDB 和 Redis 还是玩具，Go 语言刚发布
2015#
  • Apple Watch 首次发布，iPhone 6S 最强钉子户机型发布，手机第一次能拍 4K 视频，Apple Pay 刚刚在部分地区可以被接受
  • Tesla Model X 上市
  • Windows 10 发布
  • 滴滴与快的合并
  • TensorFlow 开源，AI 正式成为下一代技术发展的主流
  • VR AR 相继成为热点
  • JavaScript ES6 刚刚发布，TypeScript 进入程序员视线

看完就觉得太恐怖，这些技术都在日新月异，但区块链许多概念却声称仍在早期……文中作者还怒斥 NFT，Web3 这些概念，其实不是 2021 年才提出的新概念，而是早就发展了几年了，但只有在 2021 年币圈牛市的时候，才变成热门。“币圈一天，人间一年”说白了只是在描述币圈的涨跌幅，人们来这个圈子也只是为了逐利，真正在乎技术的又有多少呢。投机当然也不是贬义词，但盲目投机就得不偿失了。

在这里顺便推荐北大肖臻老师的《区块链技术与应用》，讲得超级好，这是zcong对这门公开课的一些学习笔记。上层应用都是过眼云烟，学好基础永不过时，适用于各个领域。

GitHub 2021 年度报告#

这两天看了看章鱼猫世界的 2021 年度报告 The 2021 State of the Octoverse，其中开篇基本上就对比了疫情前后大家的工作方式。

另外有趣的一点是 Issue as documentation，绝大多数程序员来 GitHub 上的第一件事就是提 bug，充分说明我们程序员都是爱问问题的好孩子。

JavaScript 社区谁主沉浮#

2021 JavaScript Rising Stars 发布了，从 GitHub stars 数量对 JavaScript 社区进行盘点。我看了下，总体趋势，rust 已经很大程度渗透进了 JS 社区，现在流行一句话，“所有能被 rust 重写的 JS 库都会被重写”，看起来此言得之。

  • zx，一个命令行工具勇夺第一
  • React 和 Vue 生态依然霸主，Svelte 正在迎头赶上
  • Vite 感觉不能算 Vue 生态里的，但是又和 Vue 息息相关，是今年 stars 数增长第二快的工具。其实现在 Vite 已经被广泛使用，我只能说尤大牛通。
  • Anthony Fu 作为特邀作者讲了讲 Vue 生态的发展，他的项目 slidev 名列总榜第 7，实力强悍。感兴趣的同学可以听听 ByteTalk 邀请他录的 Podcast 跟 Anthony Fu 聊聊全职开源和他的故事

Next.js 一直是我近几年最喜欢的 JS 框架，现在 Vercel 把一众大神都招了过去，组成了全明星开发团队，所以我很期待 2022 年他们会做出什么新的东西！

与读者对话#

这期我本来还写了一部分，但是好像很难归结出一个主题，索性移到了下一期再发布。我现在产出 newsletter 的流程主要是，在一周内把读过觉得不错的文章发送到 Drafts（一个写作 APP，推荐）里，然后加一两句评论，等周末再扩充并归类总结一下。我感觉现在所看到的 newsletter 大概分三类：

  • 纯写作类
  • 纯推荐链接类（可能会附有短评）
  • 链接 + 评论类（一般是长评）

我觉得最后一种比较适合我。上周因为图片太多太大，导致我的图片云存储直接欠费了……不过收到了读者的好评说“接地气”“立场与角度对普通用户很友好”感觉还是很开心的。还有一个人直接私信我该买哪个币，我就很无语，让我想起了 Ronny Chieng 脱口秀里向老婆朋友解释自己是脱口秀演员那段，有的人就是只顾自己想干什么，而不管你说什么。
