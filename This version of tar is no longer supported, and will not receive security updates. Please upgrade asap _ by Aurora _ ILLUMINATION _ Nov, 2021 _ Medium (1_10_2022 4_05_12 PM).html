This version of tar is no longer supported, and will not receive security updates. Please upgrade asap

The solution…

Aurora
Nov 24, 2021 · 1 min read

Ever gotten the vibe to start a new project emerged from the sudden spark of an idea in your mind and when your fingertips landed on the lit keyboard, all that greeted you after running “npx create-react-app project” was, “npm WARN deprecated tar@2.2.2: This version of tar is no longer supported, and will not receive security updates. Please upgrade asap” As you would have noticed, the above was not an error message because after it came, “Creating a new React app in…”. You might want to try this if you would still like to know how to resolve the message.

Solution —

  • Run “npm uninstall -g create-react-app”
  • In various situations whereby the above does not work, run “which create-react-app” and uninstall it from there
  • Then, run the command, “npx create-react-app project”

Alternatives —

  • Run “npm uninstall -g create-react-app”, then “npm cache clean --force”
  • Afterwards, try “npm cache verify” and “npx create-react-app project” or “yarn create react-app project”
